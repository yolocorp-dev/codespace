# YOLO Codespace Template

This GitHub Codespace comes with YOLO CLI pre-installed and ready to use.

## Getting Started

1. **Login**: `yolo login`
2. **Enroll in a project**: `yolo enroll`
3. **Navigate to project**: `cd <project-directory>`
4. **Run tests**: `yolo deploy`
